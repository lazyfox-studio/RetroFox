# Build
## Required
The following libraries are required by RetroFox:
* python3
* sdl2
* sdl2-ttf
* sqlite3
* qt5

## Windows
You need Microsoft Visual Studio 2019 or more recent.
Dependences for RetroFox on Windows are hangled via [vcpkg](https://github.com/Microsoft/vcpkg/).
